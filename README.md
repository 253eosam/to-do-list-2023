# To do list 2023

![vercel](https://vercelbadge.vercel.app/api/253eosam/to-do-list-2023)

투 두 리스트를 만들면서 프레임워크의 사용과 여러 기술들을 적용시켜보기 위한 프로젝트.

## ▶️ Run

> Require nodejs version 18.

### Install

Install [pnpm](https://pnpm.io/ko/installation)

```bash
$ npm i -g pnpm
```

Completed install.

```bash
$ pnpm i
```

### Dev

```bash
$ pnpm run dev
```
